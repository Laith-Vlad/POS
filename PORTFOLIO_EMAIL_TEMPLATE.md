## Tips for Sending:

1. **Customize the greeting** - Always use the recipient's name if possible
2. **Keep it concise** - Busy recruiters appreciate brevity
3. **Include metrics** - If you have any (e.g., "500+ lines of TypeScript", "15+ features implemented")
4. **Show enthusiasm** - Let your passion for the project shine through
5. **Call to action** - End with what you want (interview, feedback, discussion)
6. **Professional email** - Use a professional email address (not Gmail/Yahoo if possible)
7. **Attach resume** - If appropriate, attach your resume as PDF
8. **Follow up** - If no response, follow up after 1-2 weeks
